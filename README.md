# YOLOv8 Object Detection

Реализация системы детектирования объектов на основе YOLOv8 (You Only Look Once version 8).

## Описание

Проект предоставляет инструменты для обнаружения и классификации объектов в реальном времени с использованием предобученной модели YOLOv8. Поддерживается обработка изображений, видеопотоков и видеофайлов.

## Требования

- Python 3.8+
- CUDA-совместимая видеокарта (рекомендуется для ускорения)

## Установка

```bash
# Клонирование репозитория
git clone https://github.com/SalahidinAI/Yolo8ObjectDetection.git
cd Yolo8ObjectDetection

# Установка зависимостей
pip install -r requirements.txt
```

## Использование

### Детектирование на изображении
```python
from ultralytics import YOLO

model = YOLO('yolov8n.pt')
results = model('image.jpg')
results[0].show()
```

### Детектирование на видео
```python
model = YOLO('yolov8n.pt')
results = model('video.mp4', stream=True)

for result in results:
    result.show()
```

### Детектирование с веб-камеры
```python
model = YOLO('yolov8n.pt')
results = model(source=0, show=True)
```

## Модели

Проект поддерживает следующие варианты YOLOv8:

- **YOLOv8n** (nano) - самая быстрая, минимальная точность
- **YOLOv8s** (small) - баланс скорости и точности
- **YOLOv8m** (medium) - средняя модель
- **YOLOv8l** (large) - высокая точность
- **YOLOv8x** (extra large) - максимальная точность

## Параметры

Основные параметры конфигурации:

- `conf`: порог уверенности (по умолчанию 0.25)
- `iou`: порог IoU для NMS (по умолчанию 0.7)
- `imgsz`: размер входного изображения (по умолчанию 640)
- `device`: устройство для вычислений ('cpu' или 'cuda')

## Результаты

Модель способна детектировать 80 классов объектов из датасета COCO:
- Люди
- Транспортные средства (автомобили, мотоциклы, автобусы)
- Животные (кошки, собаки, лошади)
- Предметы быта и многое другое
